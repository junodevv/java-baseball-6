# 기능 구현 목록

1. 컴퓨터가 임의의 수 3개를 선택한다.

- 1~9까지의 서로다른 임의의 수 3개

2. 게임 플레이어에게 3개의 숫자를 입력받는다.

- 1~9까지의 서로다른 임의의 수 3개
    - 0이 포함된 경우 -> 예외
    - 숫자의 갯수 != 3 -> 예외
    - 3개의 숫자 중 같은 숫자가 있는 경우 -> 예외
        - ex) 111, 232, 988 ...
- 숫자를 입력하지 않은 경우 -> 예외
-

3. 입력된 숫자에 대한 결과를 출력한다.

- 같은 수 같은 자리 -> 스트라이크
- 같은 수 다른 자리 -> 볼
- 같은 수 없음 -> 낫싱
- 정답인 경우
    - 3스트라이크 -> 게임종료
        - 1 -> 새게임 시작
        - 2 -> 종료
        - 3이상 -> 예외
- 정답이 아닌 경우
    - 힌트 출력(ex 1스트라이크 1볼) -> 다시 숫자 입력 받기

4. 반복
